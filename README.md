#Arrays em JS
- Referente à aula do dia 19/03/26, este código faz parte de uma atividade prática de Arrays em JavaScript.
- A proposta é mexer com uma lista, uso de .pop e .push para gerenciá-la, adicionar e retirar elementos da lista como desejar.
- O estilo do CSS usado foi inspirado no Breeze Dark Theme e no Arc Dark Theme.

## Tecnologias usadas
- Ambiente de Desenvolvimento: Arch Linux (SO), VS Code (IDE), Firefox (Internet Browser)
- Linguagens: CSS e JavaScript (linguagens de programação) e HTML (linguagem de hipertexto)
